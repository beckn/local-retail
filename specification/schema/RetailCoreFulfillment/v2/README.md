# Retail Core Fulfillment Attributes (v2)

## Overview
RetailCoreFulfillmentAttributes extends Beckn core Fulfillment
for retail and logistics use cases.

## Covers
- Supported fulfillment types
- Delivery endpoint (address + contact)
- Recurring operating schedule
- Explicit closure overrides
- Lifecycle-based SLA semantics
- Handling constraints

## Does NOT duplicate
- trackingAction (core)
- fulfillmentType (core)
- start/end timestamps (core Fulfillment)

## Attachment
- Container: beckn:Fulfillment
- Field: beckn:fulfillmentAttributes
