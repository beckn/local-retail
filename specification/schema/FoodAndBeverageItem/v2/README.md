# Food & Beverage Item Attributes (v2)

## Overview

`FoodAndBeverageItemAttributes` extends `RetailCoreItemAttributes` for restaurant and prepared food use cases.

This pack captures allergens, cuisine classification, and preparation guidance while inheriting all common retail constructs.

## Attachment

- Container: `beckn:Item`
- Field: `beckn:itemAttributes`
- Extends: `RetailCoreItemAttributes`

## Covers

- Allergens
- Cuisine classification
- Preparation and storage guidance

## Does NOT Duplicate

Inherited from RetailCoreItem:
- Identity and origin
- Physical attributes
- Food regulatory declarations
- Additives
- Credentials
